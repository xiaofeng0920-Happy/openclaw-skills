## 🔧 获取真实数据的方法

### 方法 1：Tushare Pro（推荐 A 股）

**网址**: https://tushare.pro  
**费用**: 免费 + 付费（¥200-2000/年）

```python
import tushare as ts

# 初始化
ts.set_token('your_token')
pro = ts.pro_api()

# 获取日线数据
df = pro.daily(ts_code='000001.SZ', start_date='20230101', end_date='20260326')

# 获取财务数据
df = pro.fina_indicator(ts_code='000001.SZ', start_date='20230101', end_date='20260326')
```

### 方法 2：AkShare（免费）

**GitHub**: https://github.com/akfamily/akshare

```python
import akshare as ak

# A 股历史行情
df = ak.stock_zh_a_hist(symbol="000001", period="daily", start_date="20230101", end_date="20260326")

# 港股历史行情
df = ak.stock_hk_daily(symbol="00700", period="daily")
```

### 方法 3：富途 OpenD（港股实时 + 历史）

```python
from futu import *

quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)

# 获取历史 K 线

    code='HK.00700',
    start='2023-01-01',
    end='2026-03-26',
    ktype=KLType.K_DAY

```
