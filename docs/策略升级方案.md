## 🤖 方案 2：机器学习增强（1-2 周）⏳

### 目标
使用 LightGBM 模型预测股票收益率，替代线性评分。

### 实施步骤

#### 1. 数据准备（第 1-3 天）

**训练数据**:
- 过去 5 年 A 股/港股日线数据
- 15-20 个因子（动量、价值、质量、成长、情绪）
- 目标变量：未来 20 日收益率

**数据源**:
- 富途 OpenD（实时数据）
- Tushare/AkShare（历史数据）

#### 2. 模型训练（第 4-7 天）

**LightGBM 配置**:
```python
lgb_params = {
    'objective': 'regression',
    'metric': 'mse',
    'boosting_type': 'gbdt',
    'num_leaves': 31,
    'learning_rate': 0.05,
    'feature_fraction': 0.8,
    'bagging_fraction': 0.8,
    'bagging_freq': 5,
    'verbose': -1
}
```

**训练流程**:
1. 特征工程（标准化、缺失值处理）
2. 训练集/验证集划分（80%/20%）
3. 5 折交叉验证
4. 超参数调优（Grid Search）
5. 模型评估（IC、Rank IC）

#### 3. 回测验证（第 8-10 天）

**回测配置**:
- 回测期：2021-2026（5 年）
- 调仓频率：每周
- 交易成本：0.1%
- 滑点：0.2%

**对比基准**:
- 线性评分模型（现有）
- 沪深 300 / 恒生指数

#### 4. 实盘集成（第 11-14 天）

**集成方式**:
```python
# 每日预测
predictions = model.predict(today_features)

# 选择 Top N
top_stocks = predictions.sort_values(ascending=False).head(15)

# 组合优化
weights = optimize_portfolio(top_stocks)
```

#### 5. 预期效果

| 指标 | 线性模型 | LightGBM | 提升 |
|------|----------|----------|------|
| **IC** | 0.06 | 0.08-0.10 | +33-67% |
| **年化收益** | 15% | 20-25% | +33-67% |
| **夏普比率** | 0.65 | 0.85-1.0 | +31-54% |
| **最大回撤** | -20% | -15% | -25% |
