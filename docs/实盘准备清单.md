## 🛠️ 实盘操作步骤

### 步骤 1：准备账户

1. 确认富途账户已开通
2. 确认账户有足够资金（建议 10-20 万）
3. 确认交易权限（港股/美股/A 股）

### 步骤 2：启动 OpenD

1. 打开富途 OpenD GUI
2. 登录账户
3. 确认端口 11111 正常
4. 如需自动交易，解锁交易密码

### 步骤 3：首次建仓

1. 判断当前市场状态
2. 选择对应股票池
3. 生成目标持仓配置
4. 执行买入指令（分批建仓）

### 步骤 4：设置监控

1. 设置净值监控（每日）
2. 设置回撤预警（-10% 预警，-15% 减仓）
3. 设置个股止损（-12% 自动卖出）

### 步骤 5：定期调仓

1. 月度调仓：每月最后一个交易日
2. 季度调仓：每季度最后一个交易日
3. 根据市场状态调整股票池
