## 🚀 快速开始

### 1. 量化选股

```bash
cd ~/.openclaw/skills/港股量化投资

# 港股选股 Top 20
python3 scripts/quant_stock_selector.py --market HK --top 20

# 保存到指定文件
python3 scripts/quant_stock_selector.py --market HK --top 20 \
  --output reports/stock_selection_$(date +%Y%m%d).csv
```

**输出**: `reports/stock_selection_HK_YYYYMMDD.csv`

---

### 2. 组合优化

```bash
python3 scripts/portfolio_optimizer.py \
  --input reports/stock_selection_HK_20260326.csv \
  --top 15 \
  --output reports/portfolio_HK_20260326.csv
```

**输出**: `reports/portfolio_HK_YYYYMMDD.csv`

---

### 3. 模拟盘测试

```bash
python3 scripts/simulation_trading.py
```

**输出**: `reports/simulation_orders_YYYYMMDD.json`

**注意**: 模拟模式，不实际下单

---

### 4. 实盘交易

```bash
python3 scripts/real_trading.py
```

**前置条件**:
1. OpenD GUI 界面解锁交易密码
2. 账户有足够资金

**输出**: `reports/real_orders_YYYYMMDD_HHMMSS.json`

---

### 5. 组合监控

```bash
# 手动运行
python3 scripts/portfolio_monitor.py

# 定时任务（每周五 5PM）
crontab -e
# 添加：0 17 * * 5 python3 ~/.openclaw/skills/港股量化投资/scripts/portfolio_monitor.py
```

**输出**: `reports/monitoring_YYYYMMDD.md`

## 📊 选股策略

### 因子权重

| 因子 | 权重 | 子因子 |
|------|------|--------|
| **量价因子** | 60% | 20 日涨幅 50% + RSI 30% + 成交量 20% |
| **价值因子** | 40% | PE 50% + PB 30% + 股息率 20% |

### 筛选条件

- 市值：> 40 亿港元
- PE：< 30
- 价格：1-1000 港元
- 成交量：> 50 万

---

## ⚙️ 配置选项

### 修改因子权重

编辑 `config/quant_config.json`:

```json
{
  "scoring": {
    "value_weight": 0.4,    // 价值因子权重
    "momentum_weight": 0.6  // 量价因子权重
  }
}
```

### 修改筛选条件

```json
{
  "filters": {
    "min_market_cap": 50,   // 最小市值 50 亿
    "max_pe": 25,           // 最大 PE 25
    "min_price": 2          // 最低价格 2 港元
  }
}
```

## ⚠️ 故障排除

### OpenD 未连接

**错误**: `❌ OpenD 连接失败！`

**解决**:
```bash
# 检查 OpenD 状态
openclaw gateway status

# 重启 OpenD
openclaw gateway restart
```

### 交易未解锁

**错误**: `unlock needed`

**解决**:
1. 打开 OpenD GUI 界面
2. 点击「解锁交易」
3. 输入交易密码

### 数据获取失败

**错误**: `⚠️ 获取 XXX 历史数据失败`

**原因**: 港股行情权限不足

**解决**: 开通 LV1 及以上行情权限

## 📝 最佳实践

### 1. 先用模拟盘测试

建议先用模拟盘运行 3 个月，验证策略有效性。

### 2. 小资金实盘

模拟盘验证后，用 10 万港元小资金开始实盘测试。

### 3. 定期再平衡

每季度进行一次组合再平衡，恢复目标权重。

### 4. 严格止损

单股 -10% 坚决止损，不抱侥幸心理。

### 5. 长期视角

量化策略需要时间验证，建议至少持有 1 年。
