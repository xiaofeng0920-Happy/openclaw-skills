## 📋 技能使用示例

### 示例 1：判断当前市场状态
```python
from feng_quant_invest import MarketStateClassifier
import pandas as pd

# 准备数据
price_data = pd.DataFrame({
    'close': hs300_prices  # 沪深 300 历史价格
})
rsi = 58.5  # 当前 RSI
breadth = 0.68  # 市场广度

# 判断市场状态
classifier = MarketStateClassifier()
state = classifier.classify(price_data, rsi, breadth)

print(f"当前市场状态：{state}")
# 输出：当前市场状态：牛市
```

### 示例 2：生成目标持仓
```python
from feng_quant_invest import PortfolioConstructor

constructor = PortfolioConstructor()
portfolio = constructor.generate_target_portfolio(
    market_state='牛市',
    stock_pool=stock_list,  # 177 支股票
    total_capital=10000000  # 1000 万
)

print(f"目标仓位：{portfolio['target_position_ratio']:.1%}")
print(f"股票数量：{portfolio['target_stock_count']}")
```

### 示例 3：运行完整回测
```python
from feng_quant_invest import DynamicAllocationBacktest, PerformanceAnalyzer

# 运行回测
backtest = DynamicAllocationBacktest(
    start_date='2016-03-28',
    end_date='2026-03-28',

)

result = backtest.run_backtest(market_data, stock_data)

# 分析绩效
analyzer = PerformanceAnalyzer()

)

print(report)
```
